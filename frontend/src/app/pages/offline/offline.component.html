Play {{game.name}} Offline

This page will show you step-by-step how to play {{game.name}} offline on your own computer. This guide will show you how to do this using IntelliJ 2021.1. These instructions work on Windows, Mac, and Linux.

First make sure you have Java JRE and JDK installed. To install the JRE, go to https://www.java.com/en/download/manual.jsp in your web browser. Navigate to the section of the page relevant to your operating system, and select one of listed the installers. Advice exists on this page to help you decide which one to select, and how to execute your chosen installer.

To install the JDK, go to https://www.oracle.com/java/technologies/javase-jdk16-downloads.html in your browser. Again select the installer which is appropriate for your operating system, and execute this installer.

Once you have installed the JRE and the JDK, make sure you add the "java" and "javac" executables to your system path.

Running In IntelliJ

In IntelliJ, select File → New → Project... to open the "New Project" menu. Here, Select "Java" from the left panel, and set the Project SDK to your Java installation.

Click "Next" twice, and then give the project any name you'd like, and choose where you'd like to save it.

Then, click "Finish". You will see that a project is created with a folder called "src".

Click here to download a ZIP file called "{{game.id}}.zip". Extract the contents of this ZIP file (which should include a directory called "logic") into the "src" directory of your project. This can be done by right-clicking on "src" in IntelliJ and then selecting Open In → Explorer if you're on Windows, Open In → Finder if you're on Mac, and Open In → Files if you're on Linux.

As well as the "logic" folder, create two additional packages in the "src" directory named "{{game.package1}}" and "{{game.package2}}". Do this by right-clicking on "src" in IntelliJ and selecting New → Package, entering the package name, and pressing enter.

You can now put the code for two {{game.playerPlural}} into these respective packages. You can find this code by clicking here, selecting the {{game.playerSingular}} whose code you want to download, and clicking the download icon.

This will download the code as a ZIP archive. Extract its contents into either of the two packages.

You can either choose another {{game.playerSingular}} to put in the other package, or you can click here to download a template {{game.playerSingular}} which lets you {{game.offlineControls}}.

IMPORTANT: Make sure that in every Java file you add to the project, you add a line to the top which says either package {{game.package1}}; or package {{game.package2}}; depending on which package you put it into.

Next you must set the command line arguments. In IntelliJ this can be done by clicking "Program" in the top right, and then "Edit Configuration". In the "Build and run" section there is a long text box in which you can enter command line parameters separated by spaces.

{{game.params}}

With these set, click "Apply", and then click the green triangle to build and run the project.
